## 使い方
- このリポジトリの中の4つのファイル，ListPool.cs,ObjectPool.cs,Subdivide.cs,WaveMeshEffect.csを自分のプロジェクトにコピーする．
- WaveMeshEffectコンポーネントを波エフェクトさせたいImageなどにつけて，コンポーネントのインスペクタからWaveInfoを編集する．
